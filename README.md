# Hotel-Management
Phần mềm giúp quản lý, thực hiện các nghiệp vụ của một khách sạn.
## Nội dung

## Đối tượng hướng đén
- Trưởng bộ phận lễ tân
- Quản phòng khách sạn
- Trợ lý trưởng bộ phận lễ tân
- Giám sát bộ phận lễ tân
- Nhân viên lễ tân
- Nhân viên đặt phòng
- Nhân viên hỗ trợ khách hàng
